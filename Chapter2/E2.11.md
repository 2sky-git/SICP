# practice 2.11
> 分析区间的端点, 区间乘法可分解为9种情况 每种情况中乘法不超过两次   
idea: 因为乘法中需要算出4个数的最大及最小值, 在这里耗费了很大的performance

## 16种情况
(a b)*(c d). 其中a<b,c<d. 且是分配律 ac, ad, bc, bd
符合条件的可能性如下:
1. (+ +)*(+ +) => [ac,bd]
2. (+ +)*(- +) => [bc,bd]
3. (+ +)*(- -) => [bc,ad]
4. (- +)*(+ +) => []
5. (- +)*(- +) => []
6. (- +)*(- -) => []
7. (- -)*(+ +) => []
8. (- -)*(- +) => []
9. (- -)*(- -) => []
